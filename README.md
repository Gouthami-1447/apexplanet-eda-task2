# 📊 Exploratory Data Analysis (EDA) & Business Intelligence – Task 2

This repository contains Task 2 of my **Data Analytics Internship at ApexPlanet Software Pvt. Ltd.**.  
The objective of this task is to analyze data, uncover patterns, and generate meaningful business insights using EDA techniques and SQL.

---

## 🚀 Project Overview

In this project, I worked on:
- Performing descriptive and univariate analysis
- Creating visualizations to understand data distribution
- Writing SQL queries to answer business-related questions
- Conducting multivariate and correlation analysis
- Designing a dashboard mock-up to present key insights and KPIs

---

## 🛠️ Tools & Technologies Used

- Python 🐍
- Pandas 📊
- SQL 🗄️
- Matplotlib / Seaborn 📈
- Jupyter Notebook / Google Colab

---

## 📂 Key Steps

1. Data Understanding & Exploration  
2. Descriptive Statistics  
3. Data Visualization  
4. SQL-Based Business Analysis  
5. Correlation & Multivariate Analysis  
6. Dashboard Mock-up Creation  

---

## 📈 Key Insights

- Identified trends and patterns in the dataset  
- Discovered relationships between variables  
- Generated insights to support business decisions  
- Highlighted key performance indicators (KPIs)

A static dashboard mock-up was created to visualize key metrics and insights for better decision-making.
